# CSS Advanced Project

This project focuses on styling a web page using CSS, based on a design file provided on Figma. The goal is to replicate the design as closely as possible.

## Objectives

* Create an `index.html` file from the provided HTML code
* Style the page using CSS to match the Figma design
* Use fonts and images provided in the design file
